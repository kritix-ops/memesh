# Ploni font

Ploni is the brand font (licensed). Place the woff2 files here to activate it:

- `ploni-light-aaa.woff2` (300)
- `ploni-regular-aaa.woff2` (400)
- `ploni-demibold-aaa.woff2` (600)

Source (Memesh-hosted): https://memesh.co.il/wp-content/uploads/2025/12/

Until these files are present the app falls back to Assistant (loaded from Google
Fonts in index.html), which is a close visual match. The `@font-face` rules live
in `src/index.css`.
